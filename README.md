## 二：页面分析

## 实现功能



### 一：首页

* 三级列表
* 首页轮播
* 首页登录窗口
* 搜索框
* 登录展示用户信息
* 展示购物车数量
* 商品轮播
* 右侧展示客服信息
* 页尾链接



### 二：详情页

* 商品放大镜
* 选项卡
* 选择商品规格
* 加入商品数量
* 修改商品数量
* 收藏商品
* 商品详情展示

### 三：列表页

* 列表选项卡
* 列表商品分页
* 商品列表展示
* 商品右侧展示
* 商品筛选

### 四：登录页

* 用户密码登录

* 手机号验证码登录

* 下次自动登录

  

### 五：注册页

* 用户注册
* 存储信息到localstorage

### 六：购物车页

* 商品信息的展示
* 商品信息的添加
* 商品信息的删除
* 商品信息的收藏展示
* 商品信息的修改
* 全选结算
